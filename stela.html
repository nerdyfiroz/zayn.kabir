Stela Network — Project Details

Overview

Stela Network is a mobile-first crypto mining ecosystem that allows users to mine STC tokens directly from their smartphones. The project aims to make blockchain earning simple, accessible, and available to everyone while building a strong global miner community.

My Role: Community Manager

  • Managing and growing a worldwide Web3 community.
  • Onboarding thousands of new users into the Stela ecosystem.
  • Running engagement activities, campaigns, and referral-based growth systems.
  • Providing user support and resolving issues quickly.
  • Working closely with the core team to implement community-driven improvements.

Key Features of Stela Network

  • Mining via smartphone — no expensive hardware required.
  • Boosters and referrals to increase mining speed.
  • Conversion plan: STC → $STC once the network reaches 1M miners.
  • A growing global community with daily active participation.

Impact

My role strengthens user engagement, improves retention, and ensures smooth communication between the community and the project team. This helps grow a trusted, active, and informed global miner base.
