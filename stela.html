Stela Network — Project Details

Community Manager • Growth • Engagement • Web3 Ecosystem

Overview

Stela Network is a mobile-first crypto mining ecosystem that allows users to mine STC tokens directly from their smartphones. The project focuses on simplicity, accessibility, and building a large global crypto mining community.

My Role: Community Manager

  • Managing and growing a worldwide Web3 community.
  • Onboarding new users into the Stela ecosystem.
  • Running events, growth campaigns, and referral systems.
  • Providing user support and educational guidance.
  • Collaborating with the core team to optimize engagement.

Key Features of Stela Network

  • Mobile-only mining — no hardware needed.
  • Boosters & referral models increase earnings.
  • STC → $STC conversion planned at 1M miners.
  • Fast-growing global user base.
